# Spec: 萨满图腾增强系统

## 需求: 高级图腾技能

萨满祭司应拥有通过天赋解锁的高级图腾技能。

### 场景: 法力之潮图腾 (恢复系T4)
- **当** 萨满祭司投资恢复天赋至第4层
- **那么** 解锁"法力之潮图腾"技能
- **并且** 放置后每回合为全体队友恢复法力
- **并且** 持续4回合
- **并且** 冷却时间为10回合

### 场景: 大地之盾 (恢复系T5)
- **当** 萨满祭司投资恢复天赋至第5层
- **那么** 解锁"大地之盾"技能
- **并且** 为目标附加护盾
- **并且** 受击时有几率触发治疗
- **并且** 护盾有6次充能

## 需求: 图腾系统一致性

图腾技能应与现有图腾系统保持一致。

### 场景: 图腾类型分类
- **当** 放置图腾时
- **那么** 图腾按元素类型分类
- **并且** 火系图腾：灼热图腾
- **并且** 水系图腾：法力之潮图腾
- **并且** 土系图腾：大地之盾（护盾型）
- **并且** 风系图腾：（待添加）

### 场景: 图腾持续时间
- **当** 图腾被放置时
- **那么** 图腾有明确的持续时间
- **并且** 灼热图腾持续到战斗结束
- **并且** 法力之潮图腾持续4回合

## 需求: 增强系图腾相关天赋

增强系天赋树应包含图腾增强相关天赋。

### 场景: 图腾增强天赋
- **当** 查看增强系天赋树时
- **那么** 应有提升图腾效果的天赋
- **并且** 图腾法力消耗降低天赋
- **并且** 图腾持续时间延长天赋

## 需求: 恢复系天赋树结构

恢复系天赋树应具有完整的5层结构。

### 场景: 天赋层级
- **当** 查看恢复天赋树时
- **那么** 显示5层天赋
- **并且** T1有2个被动天赋（强化治疗波、潮汐专注）
- **并且** T2有2个天赋（图腾专注、治疗专注）
- **并且** T3有2个天赋（自然迅捷解锁、恢复专注）
- **并且** T4有2个天赋（法力之潮图腾解锁、治疗之赐）
- **并且** T5有2个天赋（大地之盾解锁、激流终极）

### 场景: 终极技能路径
- **当** 投资恢复天赋至T5时
- **那么** 大地之盾和激流为终极天赋
- **并且** 激流需要大地之盾作为前置
