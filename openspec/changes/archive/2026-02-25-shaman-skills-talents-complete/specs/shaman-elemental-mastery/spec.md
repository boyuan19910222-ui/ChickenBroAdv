# Spec: 萨满元素掌握系统

## 需求: 元素系天赋解锁技能

萨满祭司元素天赋树应提供完整的天赋解锁技能路径，从T4到T5终极天赋。

### 场景: 元素掌握 (T4)
- **当** 萨满祭司投资元素天赋至第4层
- **那么** 解锁"元素掌握"技能
- **并且** 激活后下一个法术必定暴击
- **并且** 冷却时间为10回合

### 场景: 熔岩爆裂 (T5)
- **当** 萨满祭司投资元素天赋至第5层
- **那么** 解锁"熔岩爆裂"技能
- **并且** 对有烈焰震击的目标必定暴击
- **并且** 造成高额火焰伤害

### 场景: 雷暴 (T5终极)
- **当** 萨满祭司完成元素天赋树终极路径
- **那么** 解锁"雷暴"技能
- **并且** 对所有敌人造成自然伤害
- **并且** 击退敌人1回合
- **并且** 恢复自身法力值

## 需求: 天赋树结构

元素系天赋树应具有完整的5层结构。

### 场景: 天赋层级
- **当** 查看元素天赋树时
- **那么** 显示5层天赋
- **并且** T1有2个被动天赋（对流、震撼）
- **并且** T2有2个天赋（烈焰呼唤、元素集中）
- **并且** T3有2个被动天赋（回响、风暴之眼）
- **并且** T4有2个天赋（元素掌握解锁、闪电超载）
- **并且** T5有2个天赋（熔岩爆裂解锁、雷暴终极）

### 场景: 天赋前置关系
- **当** 投资元素天赋时
- **那么** T2需要T1至少5点
- **并且** T3需要T1+T2至少10点
- **并且** T4需要前三层至少15点
- **并且** T5需要前四层至少20点

## 需求: 技能数据完整性

所有天赋解锁技能应有完整的技能定义。

### 场景: 技能定义存在
- **当** 天赋类型为unlock_skill时
- **那么** GameData.js中应存在对应id的技能定义
- **并且** 技能有conditions.requiresTalent指向天赋id

### 场景: 技能属性完整
- **当** 查看天赋解锁技能时
- **那么** 技能应有完整的伤害或治疗效果
- **并且** 技能应有正确的资源消耗
- **并且** 技能应有合理的冷却时间
- **并且** 技能应有清晰的描述文本
