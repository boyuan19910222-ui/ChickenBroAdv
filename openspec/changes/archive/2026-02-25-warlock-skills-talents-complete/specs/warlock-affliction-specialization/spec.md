# 术士痛苦专精规格

## 需求: 痛苦天赋树完整实现

### 场景: 生命虹吸技能
- **当** 术士投资痛苦树T3天赋「生命虹吸」时
- **那么** 解锁技能 `siphonLife`
- **并且** 对目标造成暗影DOT，持续6回合
- **并且** 每回合伤害的30%转化为术士生命
- **并且** 可与腐蚀术、痛苦诅咒同时存在

### 场景: 痛苦无常技能
- **当** 术士投资痛苦树T3天赋「痛苦无常」时
- **那么** 解锁技能 `unstableAffliction`
- **并且** 对目标施加痛苦无常DOT，持续5回合
- **并且** 如果被驱散，对驱散者造成相当于5倍tick伤害的暗影伤害
- **并且** 同时使驱散者沉默2回合

### 场景: 放大诅咒技能
- **当** 术士投资痛苦树T2天赋「放大诅咒」时
- **那么** 解锁技能 `amplifyCurse`
- **并且** 下一个诅咒法术效果提升50%
- **并且** 冷却时间5回合
- **并且** 持续到使用下一个诅咒

### 场景: 鬼影缠身终极技能
- **当** 术士投资痛苦树T5终极天赋「鬼影缠身」时
- **那么** 解锁技能 `haunt`
- **并且** 发射鬼影攻击目标，造成暗影伤害
- **并且** 使目标受到的所有DoT伤害提高20%，持续6回合
- **并且** 鬼影返回时术士获得相当于伤害值100%的治疗
- **并且** 冷却时间6回合

## 需求: 诅咒系统扩展

### 场景: 虚弱诅咒
- **当** 术士使用技能 `curseOfWeakness` 时
- **那么** 目标攻击力降低15%，持续5回合
- **并且** 每个目标只能有一个诅咒

### 场景: 元素诅咒
- **当** 术士使用技能 `curseOfElements` 时
- **那么** 目标受到的火焰和冰霜伤害提高10%，持续5回合
- **并且** 每个目标只能有一个诅咒

## 需求: 术士斩杀能力

### 场景: 暗影灼烧技能
- **当** 术士投资毁灭树T3天赋「暗影灼烧」时
- **那么** 解锁技能 `shadowburn`
- **并且** 对生命值低于20%的目标造成高额暗影伤害
- **并且** 如果目标在5回合内死亡，获得一个灵魂碎片
- **并且** 冷却时间3回合
