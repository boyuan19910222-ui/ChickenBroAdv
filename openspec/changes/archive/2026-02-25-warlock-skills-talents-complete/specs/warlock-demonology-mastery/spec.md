# 术士恶魔学识精通规格

## 需求: 恶魔学识天赋树完整实现

### 场景: 恶魔支配技能
- **当** 术士投资恶魔学识T2天赋「恶魔支配」时
- **那么** 解锁技能 `felDomination`
- **并且** 该技能可瞬发召唤恶魔（原需2行动点，现仅需1行动点）
- **并且** 冷却时间为10回合

### 场景: 灵魂链接技能
- **当** 术士投资恶魔学识T4天赋「灵魂链接」时
- **那么** 解锁被动技能 `soulLink`
- **并且** 术士受到的伤害有30%转移给当前恶魔
- **并且** 恶魔必须存活才能生效

### 场景: 恶魔变形终极技能
- **当** 术士投资恶魔学识T5终极天赋「恶魔变形」时
- **那么** 解锁主动技能 `metamorphosis`
- **并且** 术士变身为恶魔形态，持续3回合
- **并且** 恶魔形态下护甲+50%、伤害+20%、每回合回复5%生命
- **并且** 冷却时间为8回合

## 需求: 黑暗契约法力管理

### 场景: 黑暗契约技能
- **当** 术士投资痛苦树T4天赋「黑暗契约」时
- **那么** 解锁技能 `darkPact`
- **并且** 从当前恶魔抽取20%法力值转移给术士
- **并且** 不消耗行动点，冷却时间3回合
- **并且** 恶魔法力不足时无法使用

## 需求: 高级恶魔召唤

### 场景: 天赋解锁高级恶魔
- **当** 术士投资恶魔学大师天赋时
- **那么** 召唤恶魔技能可召唤更强大的恶魔类型
- **并且** 地狱猎犬提供近战DPS和隐身能力
- **并且** 魅魔提供控制能力（魅惑）
