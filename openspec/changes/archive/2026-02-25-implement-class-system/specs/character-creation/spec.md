## ADDED Requirements

### Requirement: 9职业选择界面
角色创建界面 SHALL 显示全部9个可选职业供玩家选择。

#### Scenario: 显示职业列表
- **WHEN** 玩家进入角色创建界面
- **THEN** 显示9个职业的选择按钮或卡片
- **AND** 每个职业显示名称、图标(emoji)和简短描述

#### Scenario: 职业分类展示
- **WHEN** 职业列表渲染时
- **THEN** 职业按角色定位分组显示
- **AND** 坦克职业（战士、圣骑士）标记为🛡️
- **AND** DPS职业（猎人、盗贼、法师、术士）标记为⚔️
- **AND** 治疗职业（牧师）标记为💚
- **AND** 混合职业（萨满祭司、德鲁伊）标记为🔄

### Requirement: 职业详情预览
玩家 SHALL 能够预览选中职业的详细信息。

#### Scenario: 查看职业详情
- **WHEN** 玩家选中某个职业
- **THEN** 显示该职业的详细信息面板
- **AND** 包含职业描述、角色定位、装备类型限制

#### Scenario: 预览职业属性
- **WHEN** 玩家查看职业详情
- **THEN** 显示该职业的基础属性数值
- **AND** 显示属性雷达图或条形图

#### Scenario: 预览天赋树分支
- **WHEN** 玩家查看职业详情
- **THEN** 显示该职业的3个天赋树分支名称和简述
- **AND** 提示玩家可在游戏中自由分配天赋

### Requirement: 职业特色机制提示
角色创建界面 SHALL 展示职业特色机制。

#### Scenario: 显示特殊机制
- **WHEN** 玩家选择具有特殊机制的职业
- **THEN** 显示该职业的特色机制说明
- **AND** 猎人显示"宠物系统"提示
- **AND** 术士显示"恶魔召唤"提示
- **AND** 萨满祭司显示"图腾系统"提示
- **AND** 德鲁伊显示"变形系统"提示

#### Scenario: 资源类型说明
- **WHEN** 玩家选择职业时
- **THEN** 显示该职业使用的资源类型
- **AND** 战士显示"怒气"
- **AND** 盗贼显示"能量"
- **AND** 其他职业显示"法力"

### Requirement: 职业选择确认
玩家 SHALL 能够确认职业选择并创建角色。

#### Scenario: 确认创建角色
- **WHEN** 玩家填写角色名并选择职业后点击确认
- **THEN** 系统使用选中职业的配置创建新角色
- **AND** 角色继承该职业的 baseStats 和初始 skills

#### Scenario: 职业选择验证
- **WHEN** 玩家尝试创建角色但未选择职业
- **THEN** 系统提示"请选择一个职业"
- **AND** 阻止角色创建

### Requirement: 新职业图标和样式
新增的4个职业 SHALL 拥有独特的视觉标识。

#### Scenario: 圣骑士视觉标识
- **WHEN** 显示圣骑士
- **THEN** 使用 🛡️ 或 ⚜️ 作为图标
- **AND** 使用金色系配色

#### Scenario: 萨满祭司视觉标识
- **WHEN** 显示萨满祭司
- **THEN** 使用 🔮 或 ⚡ 作为图标
- **AND** 使用蓝色系配色

#### Scenario: 术士视觉标识
- **WHEN** 显示术士
- **THEN** 使用 😈 或 🔥 作为图标
- **AND** 使用紫色系配色

#### Scenario: 德鲁伊视觉标识
- **WHEN** 显示德鲁伊
- **THEN** 使用 🌿 或 🐻 作为图标
- **AND** 使用绿色系配色
