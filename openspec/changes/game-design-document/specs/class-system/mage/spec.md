# 法师职业规格

## 新增需求

### 需求: 法师职业定位
法师是远程魔法DPS职业，专精于元素魔法和群体攻击，使用法力作为资源系统。

#### 场景: 法师角色定位
- **当** 玩家选择法师职业时
- **那么** 角色获得远程魔法DPS的定位
- **并且** 具有强大的群体伤害和控制能力

#### 场景: 法力资源系统
- **当** 法师使用法术时
- **那么** 消耗法力值释放各种魔法技能
- **并且** 法力值通过时间自然恢复或使用特殊技能恢复

### 需求: 法师装备系统
法师只能穿戴最轻的护甲并使用魔法武器。

#### 场景: 护甲穿戴能力
- **当** 法师尝试装备护甲时
- **那么** 只能穿戴布甲
- **并且** 布甲提供最少的护甲值但不影响施法

#### 场景: 武器使用能力
- **当** 法师尝试装备武器时
- **那么** 可以使用法杖、魔杖、匕首、单手剑作为武器
- **并且** 可以使用副手物品如法珠、魔典
- **并且** 无法使用盾牌、弓弩和重型武器

### 需求: 法师天赋系统
法师拥有三个天赋树：奥术、火焰和冰霜，分别对应不同的魔法专精。

#### 场景: 奥术天赋树
- **当** 玩家投资奥术天赋时
- **那么** 提升法力效率和奥术法术威力
- **并且** 解锁奥术飞弹、奥术爆炸等技能
- **并且** 适合持续输出和法力管理玩法

#### 场景: 火焰天赋树
- **当** 玩家投资火焰天赋时
- **那么** 提升火焰法术伤害和暴击几率
- **并且** 解锁火球术、炎爆术等高伤害技能
- **并且** 适合高爆发伤害玩法

#### 场景: 冰霜天赋树
- **当** 玩家投资冰霜天赋时
- **那么** 提升冰霜法术效果和控制能力
- **并且** 解锁暴风雪、冰枪术等控制技能
- **并且** 适合控制和生存玩法

### 需求: 法师元素魔法
法师掌握三种元素魔法：奥术、火焰和冰霜。

#### 场景: 奥术魔法
- **当** 法师使用奥术法术时
- **那么** 造成纯魔法伤害且消耗较少法力
- **并且** 奥术法术包括奥术飞弹、奥术智慧、法力护盾等

#### 场景: 火焰魔法
- **当** 法师使用火焰法术时
- **那么** 造成火焰伤害并可能点燃敌人
- **并且** 火焰法术包括火球术、炎爆术、火焰冲击等

#### 场景: 冰霜魔法
- **当** 法师使用冰霜法术时
- **那么** 造成冰霜伤害并减缓敌人速度
- **并且** 冰霜法术包括冰箭术、暴风雪、冰甲术等

### 需求: 法师群体攻击
法师拥有强大的群体攻击能力。

#### 场景: 范围法术
- **当** 法师使用范围法术时
- **那么** 可以同时攻击多个敌人
- **并且** 范围法术包括暴风雪、火焰风暴、奥术爆炸等

#### 场景: 连锁法术
- **当** 法师使用连锁法术时
- **那么** 法术效果可以在多个目标间跳跃传播
- **并且** 每次跳跃威力递减

### 需求: 法师传送门系统
法师拥有独特的传送和传送门能力。

#### 场景: 传送技能
- **当** 法师使用传送技能时
- **那么** 可以瞬间移动到主要城市
- **并且** 不同等级解锁不同城市的传送

#### 场景: 传送门技能
- **当** 法师创建传送门时
- **那么** 队友可以通过传送门快速到达目标城市
- **并且** 传送门有持续时间限制

### 需求: 法师魔法食物
法师可以创造魔法食物和饮料。

#### 场景: 造食术
- **当** 法师使用造食术时
- **那么** 创造魔法面包恢复生命值
- **并且** 魔法面包的恢复效果随等级提升

#### 场景: 造水术
- **当** 法师使用造水术时
- **那么** 创造魔法水恢复法力值
- **并且** 魔法水的恢复效果随等级提升

### 需求: 法师属性需求
法师主要依赖智力和精神属性。

#### 场景: 主属性影响
- **当** 法师提升智力属性时
- **那么** 增加法力值上限、法术威力和法术暴击几率
- **并且** 智力是法师最重要的属性

#### 场景: 副属性影响
- **当** 法师提升精神属性时
- **那么** 增加法力恢复速度
- **当** 法师提升耐力属性时
- **那么** 增加生命值上限，提升生存能力
