# 牧师职业规格

## 新增需求

### 需求: 牧师职业定位
牧师是主要的治疗职业，也可以担任魔法DPS角色，使用法力作为资源系统。

#### 场景: 牧师角色定位
- **当** 玩家选择牧师职业时
- **那么** 角色获得治疗和魔法DPS的双重定位
- **并且** 主要职责是为队友提供治疗和增益效果

#### 场景: 法力资源系统
- **当** 牧师使用法术时
- **那么** 消耗法力值释放治疗法术和攻击法术
- **并且** 法力值通过时间自然恢复

### 需求: 牧师装备系统
牧师只能穿戴轻甲并使用魔法武器。

#### 场景: 护甲穿戴能力
- **当** 牧师尝试装备护甲时
- **那么** 只能穿戴布甲
- **并且** 布甲提供最少的护甲值但不影响施法

#### 场景: 武器使用能力
- **当** 牧师尝试装备武器时
- **那么** 可以使用法杖、魔杖、匕首、单手锤作为武器
- **并且** 可以使用副手物品如法珠、魔典
- **并且** 无法使用剑、斧、弓弩和重型武器

### 需求: 牧师天赋系统
牧师拥有三个天赋树：神圣、戒律和暗影，分别对应治疗、防护和伤害专精。

#### 场景: 神圣天赋树
- **当** 玩家投资神圣天赋时
- **那么** 提升治疗法术效果和治疗效率
- **并且** 解锁强力治疗技能和群体治疗
- **并且** 适合主治疗师玩法

#### 场景: 戒律天赋树
- **当** 玩家投资戒律天赋时
- **那么** 提升法力效率和防护能力
- **并且** 解锁伤害吸收和增益法术
- **并且** 适合辅助治疗和团队支援玩法

#### 场景: 暗影天赋树
- **当** 玩家投资暗影天赋时
- **那么** 提升暗影法术伤害和法术暴击
- **并且** 解锁攻击性暗影法术
- **并且** 适合魔法DPS玩法

### 需求: 牧师治疗系统
牧师拥有最强大和全面的治疗能力。

#### 场景: 单体治疗
- **当** 牧师使用单体治疗法术时
- **那么** 可以使用治疗术快速恢复目标生命值
- **并且** 可以使用强效治疗术恢复大量生命值
- **并且** 可以使用快速治疗术进行紧急救治

#### 场景: 群体治疗
- **当** 牧师使用群体治疗法术时
- **那么** 可以使用群体治疗同时恢复多个队友生命值
- **并且** 可以使用治疗祷言为队伍提供持续治疗

#### 场景: 治疗over时间
- **当** 牧师使用持续治疗法术时
- **那么** 恢复术提供持续的生命值恢复
- **并且** 治疗效果在一段时间内持续生效

### 需求: 牧师增益系统
牧师可以为队友提供各种强力的增益效果。

#### 场景: 属性增益
- **当** 牧师施放增益法术时
- **那么** 真言术：韧提升队友耐力
- **并且** 真言术：毅提升队友精神
- **并且** 神圣之灵提升队友精神恢复

#### 场景: 防护增益
- **当** 牧师施放防护法术时
- **那么** 真言术：盾为目标提供伤害吸收
- **并且** 防护恐惧使目标免疫恐惧效果
- **并且** 驱散魔法移除目标身上的负面效果

### 需求: 牧师暗影魔法
牧师也掌握暗影系攻击法术。

#### 场景: 暗影攻击法术
- **当** 牧师使用暗影法术时
- **那么** 可以使用暗影箭造成暗影伤害
- **并且** 可以使用精神鞭笞进行持续施法攻击
- **并且** 可以使用心灵震爆造成范围暗影伤害

#### 场景: 暗影控制法术
- **当** 牧师使用控制法术时
- **那么** 可以使用心灵控制操控敌人
- **并且** 可以使用恐惧术使敌人逃跑
- **并且** 可以使用沉默术阻止敌人施法

### 需求: 牧师复活能力
牧师拥有复活死亡队友的能力。

#### 场景: 战斗外复活
- **当** 牧师在非战斗状态下使用复活术时
- **那么** 可以复活死亡的队友
- **并且** 复活需要较长的施法时间

#### 场景: 战斗中复活
- **当** 牧师学习相关天赋后
- **那么** 可以在战斗中使用瞬发复活
- **并且** 战斗复活有冷却时间限制

### 需求: 牧师种族法术
不同种族的牧师拥有独特的种族专属法术。

#### 场景: 人类牧师法术
- **当** 人类牧师达到相应等级时
- **那么** 学习绝望祷言和反馈
- **并且** 这些法术只有人类牧师可以使用

#### 场景: 矮人牧师法术
- **当** 矮人牧师达到相应等级时
- **那么** 学习恐惧守护和石肤术
- **并且** 这些法术只有矮人牧师可以使用

### 需求: 牧师属性需求
牧师根据专精方向对不同属性有不同需求。

#### 场景: 治疗专精属性
- **当** 牧师专精治疗时
- **那么** 主要需求智力和精神属性
- **并且** 智力提升法力值和治疗效果，精神提升法力恢复

#### 场景: 暗影专精属性
- **当** 牧师专精暗影时
- **那么** 主要需求智力属性，对精神需求较少
- **并且** 智力提升法力值和法术伤害

#### 场景: 生存属性
- **当** 牧师提升耐力属性时
- **那么** 增加生命值上限，提升生存能力
- **并且** 对于脆弱的布甲职业来说耐力很重要
