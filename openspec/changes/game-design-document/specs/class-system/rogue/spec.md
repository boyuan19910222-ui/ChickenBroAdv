# 盗贼职业规格

## 新增需求

### 需求: 盗贼职业定位
盗贼是近战物理DPS职业，专精于潜行、暗杀和连击，使用能量作为资源系统。

#### 场景: 盗贼角色定位
- **当** 玩家选择盗贼职业时
- **那么** 角色获得近战DPS的定位
- **并且** 具有独特的潜行能力和高爆发伤害

#### 场景: 能量资源系统
- **当** 盗贼使用技能时
- **那么** 消耗能量值释放攻击技能和特殊能力
- **并且** 能量值快速自然恢复

### 需求: 盗贼装备系统
盗贼穿戴轻甲并专精于敏捷武器。

#### 场景: 护甲穿戴能力
- **当** 盗贼尝试装备护甲时
- **那么** 可以穿戴布甲和皮甲
- **并且** 皮甲提供最佳的防护效果

#### 场景: 武器使用能力
- **当** 盗贼尝试装备武器时
- **那么** 可以使用匕首、单手剑、单手斧、拳套作为主手武器
- **并且** 可以进行双武器战斗
- **并且** 可以使用弓、弩、投掷武器作为远程武器
- **并且** 无法使用盾牌和双手武器

### 需求: 盗贼天赋系统
盗贼拥有三个天赋树：刺杀、战斗和敏锐，分别对应暗杀、双武器和潜行专精。

#### 场景: 刺杀天赋树
- **当** 玩家投资刺杀天赋时
- **那么** 提升匕首使用技巧和毒药效果
- **并且** 解锁强力暗杀技能和致命毒药
- **并且** 适合匕首暗杀玩法

#### 场景: 战斗天赋树
- **当** 玩家投资战斗天赋时
- **那么** 提升双武器战斗能力和攻击速度
- **并且** 解锁连击技能和武器专精
- **并且** 适合双武器DPS玩法

#### 场景: 敏锐天赋树
- **当** 玩家投资敏锐天赋时
- **那么** 提升潜行能力和暴击伤害
- **并且** 解锁高级潜行技能和陷阱能力
- **并且** 适合潜行控制玩法

### 需求: 盗贼潜行系统
盗贼拥有独特的潜行和隐身能力。

#### 场景: 潜行状态
- **当** 盗贼进入潜行状态时
- **那么** 角色变为半透明且难以被敌人发现
- **并且** 移动速度降低但可以绕过敌人

#### 场景: 潜行攻击
- **当** 盗贼在潜行状态下攻击时
- **那么** 第一次攻击造成额外伤害
- **并且** 可以使用特殊的开场技能如偷袭、伏击

#### 场景: 潜行技能
- **当** 盗贼在潜行状态下时
- **那么** 可以使用扒窃技能偷取敌人物品
- **并且** 可以使用闷棍技能昏迷敌人

### 需求: 盗贼连击系统
盗贼使用连击点数系统来释放强力终结技。

#### 场景: 连击点积累
- **当** 盗贼使用起手技能时
- **那么** 获得1-2个连击点数
- **并且** 最多可以积累5个连击点数

#### 场景: 终结技释放
- **当** 盗贼使用终结技时
- **那么** 消耗所有连击点数造成伤害或效果
- **并且** 连击点数越多，终结技效果越强

#### 场景: 连击技能类型
- **当** 盗贼使用不同连击技能时
- **那么** 邪恶攻击、背刺、出血为起手技能
- **并且** 剔骨、切割、肾击为终结技能

### 需求: 盗贼毒药系统
盗贼可以在武器上涂抹各种毒药增强攻击效果。

#### 场景: 毒药制作
- **当** 盗贼制作毒药时
- **那么** 使用毒药材料和毒药技能制作不同类型的毒药
- **并且** 毒药有使用次数限制

#### 场景: 毒药涂抹
- **当** 盗贼在武器上涂抹毒药时
- **那么** 武器攻击有几率触发毒药效果
- **并且** 不同毒药有不同的效果和持续时间

#### 场景: 毒药类型
- **当** 毒药生效时
- **那么** 致命毒药造成持续伤害
- **并且** 致伤毒药降低敌人治疗效果
- **并且** 麻痹毒药减缓敌人攻击和移动速度

### 需求: 盗贼属性需求
盗贼主要依赖敏捷属性，对力量也有一定需求。

#### 场景: 主属性影响
- **当** 盗贼提升敏捷属性时
- **那么** 增加攻击强度、暴击几率、躲闪几率和护甲值
- **并且** 敏捷是盗贼最重要的属性

#### 场景: 副属性影响
- **当** 盗贼提升力量属性时
- **那么** 增加近战攻击强度
- **当** 盗贼提升耐力属性时
- **那么** 增加生命值上限，提升生存能力
