# 战士职业规格

## 新增需求

### 需求: 战士职业定位
战士是主要的坦克职业，专精于近战战斗和承受伤害，使用怒气作为资源系统。

#### 场景: 战士角色定位
- **当** 玩家选择战士职业时
- **那么** 角色获得坦克和近战DPS的双重定位
- **并且** 主要职责是保护队友和承受敌人攻击

#### 场景: 怒气资源系统
- **当** 战士进行战斗时
- **那么** 通过攻击和受到攻击积累怒气值
- **并且** 消耗怒气值释放强力技能

### 需求: 战士装备系统
战士可以穿戴最重的护甲并使用多种武器类型。

#### 场景: 护甲穿戴能力
- **当** 战士尝试装备护甲时
- **那么** 可以穿戴布甲、皮甲、锁甲和板甲
- **并且** 板甲提供最高的护甲值和防御能力

#### 场景: 武器使用能力
- **当** 战士尝试装备武器时
- **那么** 可以使用单手剑、双手剑、单手斧、双手斧、单手锤、双手锤、匕首、长柄武器、弓、弩和盾牌
- **并且** 可以进行双武器战斗或武器盾牌组合

### 需求: 战士天赋系统
战士拥有三个天赋树：武器、狂暴和防护，分别对应不同的专精方向。

#### 场景: 武器天赋树
- **当** 玩家投资武器天赋时
- **那么** 提升武器熟练度和战斗技巧
- **并且** 解锁武器专精技能和被动效果
- **并且** 适合双武器DPS和单手武器DPS玩法

#### 场景: 狂暴天赋树
- **当** 玩家投资狂暴天赋时
- **那么** 提升攻击速度和暴击能力
- **并且** 解锁狂暴状态和连击技能
- **并且** 适合双手武器DPS玩法

#### 场景: 防护天赋树
- **当** 玩家投资防护天赋时
- **那么** 提升防御能力和仇恨生成
- **并且** 解锁盾牌技能和防御姿态
- **并且** 适合主坦克玩法

### 需求: 战士核心技能
战士拥有多种战斗姿态和对应的技能组合。

#### 场景: 战斗姿态切换
- **当** 战士切换战斗姿态时
- **那么** 获得不同的属性加成和技能可用性
- **并且** 战斗姿态提升攻击力但降低防御力
- **并且** 防御姿态提升防御力和仇恨生成但降低攻击力
- **并且** 狂暴姿态提升暴击率但增加受到的伤害

#### 场景: 核心攻击技能
- **当** 战士使用攻击技能时
- **那么** 可以使用英勇打击、顺劈斩、雷霆一击等基础技能
- **并且** 高级技能包括致死打击、血腥撕裂、斩杀等

#### 场景: 防御技能
- **当** 战士使用防御技能时
- **那么** 可以使用盾牌格挡、盾牌猛击、嘲讽等技能
- **并且** 能够有效控制敌人仇恨和减少受到的伤害

### 需求: 战士属性需求
战士主要依赖力量和耐力属性，对敏捷也有一定需求。

#### 场景: 主属性影响
- **当** 战士提升力量属性时
- **那么** 增加攻击强度和格挡几率
- **并且** 力量是战士最重要的属性

#### 场景: 副属性影响
- **当** 战士提升耐力属性时
- **那么** 增加生命值上限，提升生存能力
- **当** 战士提升敏捷属性时
- **那么** 增加护甲值、暴击几率和躲闪几率
