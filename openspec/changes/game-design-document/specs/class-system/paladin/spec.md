# 圣骑士职业规格

## 新增需求

### 需求: 圣骑士职业定位
圣骑士是混合职业，可以担任坦克、治疗和近战DPS角色，使用法力作为资源系统。

#### 场景: 圣骑士角色定位
- **当** 玩家选择圣骑士职业时
- **那么** 角色获得坦克、治疗和近战DPS的三重定位
- **并且** 具有强大的生存能力和团队支援能力

#### 场景: 法力资源系统
- **当** 圣骑士使用技能时
- **那么** 消耗法力值释放圣印、光环和治疗法术
- **并且** 法力值通过时间自然恢复

### 需求: 圣骑士装备系统
圣骑士可以穿戴重甲并使用特定的武器类型。

#### 场景: 护甲穿戴能力
- **当** 圣骑士尝试装备护甲时
- **那么** 可以穿戴布甲、皮甲、锁甲和板甲
- **并且** 板甲提供最佳的防护效果

#### 场景: 武器使用能力
- **当** 圣骑士尝试装备武器时
- **那么** 可以使用单手剑、单手斧、单手锤、双手剑、双手斧、双手锤和盾牌
- **并且** 无法使用弓、弩、匕首、长柄武器等

### 需求: 圣骑士天赋系统
圣骑士拥有三个天赋树：神圣、防护和惩戒，分别对应治疗、坦克和DPS专精。

#### 场景: 神圣天赋树
- **当** 玩家投资神圣天赋时
- **那么** 提升治疗法术效果和法力效率
- **并且** 解锁强力治疗技能和光环效果
- **并且** 适合团队治疗师玩法

#### 场景: 防护天赋树
- **当** 玩家投资防护天赋时
- **那么** 提升防御能力和仇恨生成
- **并且** 解锁防御性圣印和盾牌技能
- **并且** 适合主坦克玩法

#### 场景: 惩戒天赋树
- **当** 玩家投资惩戒天赋时
- **那么** 提升近战攻击力和暴击能力
- **并且** 解锁攻击性圣印和惩戒技能
- **并且** 适合近战DPS玩法

### 需求: 圣骑士核心技能
圣骑士拥有独特的圣印、光环和圣化系统。

#### 场景: 圣印系统
- **当** 圣骑士激活圣印时
- **那么** 武器攻击获得额外效果
- **并且** 同时只能激活一个圣印
- **并且** 圣印包括力量圣印、智慧圣印、光明圣印等

#### 场景: 光环系统
- **当** 圣骑士激活光环时
- **那么** 为周围队友提供持续的增益效果
- **并且** 同时只能激活一个光环
- **并且** 光环包括虔诚光环、专注光环、抗性光环等

#### 场景: 圣化技能
- **当** 圣骑士使用圣化技能时
- **那么** 在地面创造神圣区域，对敌人造成持续伤害
- **并且** 站在圣化区域内的圣骑士获得额外效果

### 需求: 圣骑士治疗能力
圣骑士拥有强大的治疗和支援能力。

#### 场景: 治疗法术
- **当** 圣骑士使用治疗法术时
- **那么** 可以使用圣光术、神圣之光等单体治疗
- **并且** 治疗效果受智力属性影响

#### 场景: 保护祝福
- **当** 圣骑士施放祝福时
- **那么** 为队友提供各种保护效果
- **并且** 祝福包括力量祝福、智慧祝福、保护祝福等

### 需求: 圣骑士属性需求
圣骑士根据专精方向对不同属性有不同需求。

#### 场景: 防护专精属性
- **当** 圣骑士专精防护时
- **那么** 主要需求耐力和力量属性
- **并且** 耐力提升生命值，力量提升攻击力和格挡

#### 场景: 神圣专精属性
- **当** 圣骑士专精神圣时
- **那么** 主要需求智力和精神属性
- **并且** 智力提升法力值和治疗效果，精神提升法力恢复

#### 场景: 惩戒专精属性
- **当** 圣骑士专精惩戒时
- **那么** 主要需求力量和敏捷属性
- **并且** 力量提升攻击力，敏捷提升暴击率
