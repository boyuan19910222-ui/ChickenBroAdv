# 术士职业规格

## 新增需求

### 需求: 术士职业定位
术士是远程魔法DPS职业，专精于暗影魔法和恶魔召唤，使用法力作为资源系统。

#### 场景: 术士角色定位
- **当** 玩家选择术士职业时
- **那么** 角色获得远程魔法DPS的定位
- **并且** 具有独特的恶魔召唤和持续伤害能力

#### 场景: 法力资源系统
- **当** 术士使用法术时
- **那么** 消耗法力值释放暗影法术和召唤恶魔
- **并且** 可以通过生命分流将生命值转换为法力值

### 需求: 术士装备系统
术士只能穿戴轻甲并使用魔法武器。

#### 场景: 护甲穿戴能力
- **当** 术士尝试装备护甲时
- **那么** 只能穿戴布甲
- **并且** 布甲提供最少的护甲值但不影响施法

#### 场景: 武器使用能力
- **当** 术士尝试装备武器时
- **那么** 可以使用法杖、魔杖、匕首、单手剑作为武器
- **并且** 可以使用副手物品如法珠、魔典
- **并且** 无法使用盾牌、弓弩和重型武器

### 需求: 术士天赋系统
术士拥有三个天赋树：痛苦、恶魔学识和毁灭，分别对应DoT、召唤和直伤专精。

#### 场景: 痛苦天赋树
- **当** 玩家投资痛苦天赋时
- **那么** 提升持续伤害法术效果和持续时间
- **并且** 解锁腐蚀术、痛苦诅咒等DoT技能
- **并且** 适合持续伤害和多目标玩法

#### 场景: 恶魔学识天赋树
- **当** 玩家投资恶魔学识天赋时
- **那么** 提升恶魔召唤能力和宠物强度
- **并且** 解锁高级恶魔和召唤增强技能
- **并且** 适合依赖恶魔宠物的玩法

#### 场景: 毁灭天赋树
- **当** 玩家投资毁灭天赋时
- **那么** 提升直接伤害法术威力和暴击几率
- **并且** 解锁暗影箭、灵魂火焰等高伤害技能
- **并且** 适合高爆发伤害玩法

### 需求: 术士恶魔召唤
术士拥有独特的恶魔召唤和管理系统。

#### 场景: 恶魔召唤
- **当** 术士召唤恶魔时
- **那么** 消耗灵魂碎片召唤不同类型的恶魔
- **并且** 同时只能存在一个召唤的恶魔

#### 场景: 恶魔类型
- **当** 术士召唤不同恶魔时
- **那么** 小鬼提供远程攻击支援
- **并且** 虚空行者作为坦克保护术士
- **并且** 魅魔提供控制和诱惑能力
- **并且** 地狱猎犬提供近战攻击和隐身能力

#### 场景: 恶魔控制
- **当** 术士控制恶魔时
- **那么** 可以设置恶魔的行为模式
- **并且** 可以直接指挥恶魔攻击特定目标
- **并且** 恶魔死亡后需要重新召唤

### 需求: 术士诅咒系统
术士可以对敌人施放各种诅咒效果。

#### 场景: 诅咒施放
- **当** 术士对敌人施放诅咒时
- **那么** 敌人获得负面效果
- **并且** 每个敌人同时只能承受一个诅咒

#### 场景: 诅咒类型
- **当** 不同诅咒生效时
- **那么** 痛苦诅咒造成持续伤害
- **并且** 虚弱诅咒降低敌人攻击力
- **并且** 语言诅咒增加敌人施法时间
- **并且** 末日诅咒造成延迟的巨额伤害

### 需求: 术士持续伤害
术士专精于持续伤害（DoT）法术。

#### 场景: DoT法术施放
- **当** 术士使用DoT法术时
- **那么** 对敌人造成持续的伤害效果
- **并且** 可以同时维持多个不同的DoT效果

#### 场景: DoT法术类型
- **当** DoT法术生效时
- **那么** 腐蚀术造成持续的暗影伤害
- **并且** 献祭造成持续的火焰伤害
- **并且** 虹吸灵魂在造成伤害的同时恢复术士法力

### 需求: 术士灵魂系统
术士使用灵魂碎片作为特殊资源。

#### 场景: 灵魂碎片获取
- **当** 敌人在术士的DoT效果下死亡时
- **那么** 术士获得灵魂碎片
- **并且** 灵魂碎片存储在灵魂袋中

#### 场景: 灵魂碎片使用
- **当** 术士使用灵魂碎片时
- **那么** 可以召唤恶魔或使用特殊技能
- **并且** 可以创造治疗石和法力石

### 需求: 术士属性需求
术士主要依赖智力和精神属性，对耐力也有需求。

#### 场景: 主属性影响
- **当** 术士提升智力属性时
- **那么** 增加法力值上限、法术威力和法术暴击几率
- **并且** 智力是术士最重要的属性

#### 场景: 副属性影响
- **当** 术士提升精神属性时
- **那么** 增加法力恢复速度
- **当** 术士提升耐力属性时
- **那么** 增加生命值上限，提升生存能力
- **并且** 耐力对术士尤为重要，因为生命分流技能会消耗生命值
