# 萨满祭司职业规格

## 新增需求

### 需求: 萨满祭司职业定位
萨满祭司是混合职业，可以担任治疗、近战DPS和远程DPS角色，使用法力作为资源系统。

#### 场景: 萨满祭司角色定位
- **当** 玩家选择萨满祭司职业时
- **那么** 角色获得治疗、近战DPS和远程DPS的三重定位
- **并且** 具有独特的图腾系统和元素掌控能力

#### 场景: 法力资源系统
- **当** 萨满祭司使用技能时
- **那么** 消耗法力值释放法术、治疗和图腾
- **并且** 法力值通过时间自然恢复

### 需求: 萨满祭司装备系统
萨满祭司可以穿戴中等护甲并使用多种武器类型。

#### 场景: 护甲穿戴能力
- **当** 萨满祭司尝试装备护甲时
- **那么** 可以穿戴布甲、皮甲和锁甲
- **并且** 锁甲提供最佳的防护效果

#### 场景: 武器使用能力
- **当** 萨满祭司尝试装备武器时
- **那么** 可以使用单手斧、单手锤、法杖、匕首、拳套作为武器
- **并且** 可以使用盾牌和副手物品
- **并且** 无法使用剑、弓弩和长柄武器

### 需求: 萨满祭司天赋系统
萨满祭司拥有三个天赋树：元素、增强和恢复，分别对应远程DPS、近战DPS和治疗专精。

#### 场景: 元素天赋树
- **当** 玩家投资元素天赋时
- **那么** 提升元素法术伤害和法术暴击
- **并且** 解锁强力元素法术和自然抗性
- **并且** 适合远程魔法DPS玩法

#### 场景: 增强天赋树
- **当** 玩家投资增强天赋时
- **那么** 提升近战攻击能力和武器增强效果
- **并且** 解锁风怒武器和火舌武器等增强技能
- **并且** 适合近战DPS玩法

#### 场景: 恢复天赋树
- **当** 玩家投资恢复天赋时
- **那么** 提升治疗法术效果和法力效率
- **并且** 解锁强力治疗技能和群体治疗
- **并且** 适合团队治疗师玩法

### 需求: 萨满祭司图腾系统
萨满祭司拥有独特的图腾放置和管理系统。

#### 场景: 图腾放置
- **当** 萨满祭司放置图腾时
- **那么** 图腾在指定位置提供持续效果
- **并且** 图腾有持续时间和生命值限制
- **并且** 可以同时放置4个不同类型的图腾

#### 场景: 图腾类型分类
- **当** 萨满祭司放置不同类型图腾时
- **那么** 火系图腾提供攻击和伤害增益
- **并且** 土系图腾提供防御和抗性增益
- **并且** 水系图腾提供治疗和法力恢复
- **并且** 风系图腾提供移动和攻击速度增益

#### 场景: 具体图腾效果
- **当** 图腾生效时
- **那么** 灼热图腾对敌人造成火焰伤害
- **并且** 治疗之泉图腾为队友恢复生命值
- **并且** 风怒图腾提升队友攻击速度
- **并且** 石肤图腾提升队友护甲值

### 需求: 萨满祭司元素法术
萨满祭司掌握多种元素魔法。

#### 场景: 闪电法术
- **当** 萨满祭司使用闪电法术时
- **那么** 可以使用闪电箭进行远程攻击
- **并且** 可以使用闪电链攻击多个目标
- **并且** 闪电法术造成自然伤害

#### 场景: 火焰法术
- **当** 萨满祭司使用火焰法术时
- **那么** 可以使用火焰震击造成即时和持续伤害
- **并且** 可以使用熔岩爆裂造成高额火焰伤害

#### 场景: 冰霜法术
- **当** 萨满祭司使用冰霜法术时
- **那么** 可以使用冰霜震击减缓敌人移动
- **并且** 冰霜法术同时造成伤害和控制效果

### 需求: 萨满祭司治疗能力
萨满祭司拥有强大的治疗和群体治疗能力。

#### 场景: 单体治疗
- **当** 萨满祭司使用治疗法术时
- **那么** 可以使用治疗波快速恢复目标生命值
- **并且** 可以使用强效治疗波恢复大量生命值

#### 场景: 群体治疗
- **当** 萨满祭司使用群体治疗时
- **那么** 可以使用治疗链同时治疗多个队友
- **并且** 治疗效果在目标间跳跃传播，每次跳跃效果递减

### 需求: 萨满祭司武器增强
萨满祭司可以用元素力量增强武器。

#### 场景: 武器增强施放
- **当** 萨满祭司对武器施放增强时
- **那么** 武器获得额外的元素伤害效果
- **并且** 增强效果有持续时间限制

#### 场景: 增强类型
- **当** 不同武器增强生效时
- **那么** 火舌武器增加火焰伤害
- **并且** 风怒武器提升攻击速度和额外攻击机会
- **并且** 冰封武器减缓敌人移动速度

### 需求: 萨满祭司属性需求
萨满祭司根据专精方向对不同属性有不同需求。

#### 场景: 元素专精属性
- **当** 萨满祭司专精元素时
- **那么** 主要需求智力和精神属性
- **并且** 智力提升法力值和法术伤害

#### 场景: 增强专精属性
- **当** 萨满祭司专精增强时
- **那么** 主要需求力量和敏捷属性
- **并且** 力量提升近战攻击力，敏捷提升暴击率

#### 场景: 恢复专精属性
- **当** 萨满祭司专精恢复时
- **那么** 主要需求智力和精神属性
- **并且** 智力提升治疗效果，精神提升法力恢复
