# 猎人职业规格

## 新增需求

### 需求: 猎人职业定位
猎人是远程物理DPS职业，专精于远程攻击和宠物驯养，使用法力作为资源系统。

#### 场景: 猎人角色定位
- **当** 玩家选择猎人职业时
- **那么** 角色获得远程DPS的定位
- **并且** 具有独特的宠物伙伴和陷阱控制能力

#### 场景: 法力资源系统
- **当** 猎人使用技能时
- **那么** 消耗法力值释放射击技能和宠物指令
- **并且** 法力值通过时间自然恢复

### 需求: 猎人装备系统
猎人可以穿戴中等护甲并专精于远程武器。

#### 场景: 护甲穿戴能力
- **当** 猎人尝试装备护甲时
- **那么** 可以穿戴布甲、皮甲和锁甲
- **并且** 锁甲提供最佳的防护效果

#### 场景: 武器使用能力
- **当** 猎人尝试装备武器时
- **那么** 可以使用弓、弩、枪械作为主要武器
- **并且** 可以使用单手剑、单手斧、匕首、长柄武器作为近战武器
- **并且** 无法使用盾牌和双手武器

### 需求: 猎人天赋系统
猎人拥有三个天赋树：野兽掌握、射击和生存，分别对应宠物、远程和陷阱专精。

#### 场景: 野兽掌握天赋树
- **当** 玩家投资野兽掌握天赋时
- **那么** 提升宠物能力和驯养技巧
- **并且** 解锁高级宠物技能和特殊宠物类型
- **并且** 适合依赖宠物输出的玩法

#### 场景: 射击天赋树
- **当** 玩家投资射击天赋时
- **那么** 提升远程攻击力和射击技能
- **并且** 解锁强力射击技能和被动效果
- **并且** 适合纯远程DPS玩法

#### 场景: 生存天赋树
- **当** 玩家投资生存天赋时
- **那么** 提升陷阱效果和生存能力
- **并且** 解锁高级陷阱和近战技能
- **并且** 适合控制和辅助玩法

### 需求: 猎人宠物系统
猎人拥有独特的宠物驯养和管理系统。

#### 场景: 宠物驯养
- **当** 猎人遇到可驯养的野兽时
- **那么** 可以使用驯养技能将其收为宠物
- **并且** 不同类型的野兽有不同的特殊能力

#### 场景: 宠物管理
- **当** 猎人管理宠物时
- **那么** 可以喂养宠物提升忠诚度
- **并且** 可以训练宠物学习新技能
- **并且** 可以设置宠物的行为模式（主动、被动、防御）

#### 场景: 宠物战斗
- **当** 宠物参与战斗时
- **那么** 宠物可以使用特殊技能攻击敌人
- **并且** 宠物可以为猎人承受部分伤害
- **并且** 宠物死亡后可以通过复活技能恢复

### 需求: 猎人射击技能
猎人拥有多种远程攻击和特殊射击技能。

#### 场景: 基础射击技能
- **当** 猎人使用射击技能时
- **那么** 可以使用自动射击进行持续攻击
- **并且** 可以使用奥术射击、多重射击等特殊技能

#### 场景: 高级射击技能
- **当** 猎人使用高级技能时
- **那么** 可以使用瞄准射击造成高额伤害
- **并且** 可以使用穿透射击攻击多个目标
- **并且** 可以使用爆炸射击造成范围伤害

### 需求: 猎人陷阱系统
猎人可以放置各种陷阱来控制战场。

#### 场景: 陷阱放置
- **当** 猎人放置陷阱时
- **那么** 陷阱在指定位置等待敌人触发
- **并且** 同时只能存在有限数量的陷阱

#### 场景: 陷阱类型
- **当** 敌人触发陷阱时
- **那么** 冰霜陷阱减缓敌人移动速度
- **并且** 爆炸陷阱对敌人造成火焰伤害
- **并且** 固定陷阱使敌人无法移动

### 需求: 猎人属性需求
猎人主要依赖敏捷和智力属性。

#### 场景: 主属性影响
- **当** 猎人提升敏捷属性时
- **那么** 增加远程攻击强度、暴击几率和躲闪几率
- **并且** 敏捷是猎人最重要的属性

#### 场景: 副属性影响
- **当** 猎人提升智力属性时
- **那么** 增加法力值上限，支持更多技能使用
- **当** 猎人提升耐力属性时
- **那么** 增加生命值上限，提升生存能力
