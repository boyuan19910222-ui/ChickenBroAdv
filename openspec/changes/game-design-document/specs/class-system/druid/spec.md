# 德鲁伊职业规格

## 新增需求

### 需求: 德鲁伊职业定位
德鲁伊是最全能的混合职业，可以担任坦克、治疗、近战DPS和远程DPS角色，使用法力作为资源系统。

#### 场景: 德鲁伊角色定位
- **当** 玩家选择德鲁伊职业时
- **那么** 角色获得坦克、治疗、近战DPS和远程DPS的四重定位
- **并且** 具有独特的变形系统和自然魔法能力

#### 场景: 法力资源系统
- **当** 德鲁伊使用技能时
- **那么** 消耗法力值释放法术、治疗和变形技能
- **并且** 不同形态下可能使用不同的资源系统

### 需求: 德鲁伊装备系统
德鲁伊只能穿戴轻甲并使用特定的武器类型。

#### 场景: 护甲穿戴能力
- **当** 德鲁伊尝试装备护甲时
- **那么** 可以穿戴布甲和皮甲
- **并且** 皮甲提供最佳的防护效果

#### 场景: 武器使用能力
- **当** 德鲁伊尝试装备武器时
- **那么** 可以使用法杖、单手锤、匕首、拳套作为武器
- **并且** 可以使用副手物品如图腾、魔像
- **并且** 无法使用金属武器如剑、斧、长柄武器

### 需求: 德鲁伊天赋系统
德鲁伊拥有三个天赋树：平衡、野性战斗和恢复，分别对应远程DPS、近战/坦克和治疗专精。

#### 场景: 平衡天赋树
- **当** 玩家投资平衡天赋时
- **那么** 提升自然法术伤害和法术暴击
- **并且** 解锁强力自然法术和月火术增强
- **并且** 适合远程魔法DPS玩法

#### 场景: 野性战斗天赋树
- **当** 玩家投资野性战斗天赋时
- **那么** 提升变形能力和野兽形态战斗力
- **并且** 解锁高级变形和野兽技能
- **并且** 适合坦克和近战DPS玩法

#### 场景: 恢复天赋树
- **当** 玩家投资恢复天赋时
- **那么** 提升治疗法术效果和持续治疗
- **并且** 解锁强力治疗技能和群体治疗
- **并且** 适合团队治疗师玩法

### 需求: 德鲁伊变形系统
德鲁伊拥有独特的动物变形能力。

#### 场景: 熊形态
- **当** 德鲁伊变形为熊形态时
- **那么** 获得大量护甲值和生命值加成
- **并且** 可以使用熊形态专属技能如挥击、咆哮
- **并且** 适合担任坦克角色

#### 场景: 猫形态
- **当** 德鲁伊变形为猫形态时
- **那么** 获得潜行能力和攻击速度加成
- **并且** 可以使用猫形态专属技能如撕咬、割裂
- **并且** 适合担任近战DPS角色

#### 场景: 海豹形态
- **当** 德鲁伊变形为海豹形态时
- **那么** 获得水下呼吸和游泳速度加成
- **并且** 适合水下探索和移动

#### 场景: 旅行形态
- **当** 德鲁伊变形为旅行形态时
- **那么** 获得移动速度加成
- **并且** 适合快速移动和探索

### 需求: 德鲁伊自然魔法
德鲁伊掌握多种自然系法术。

#### 场景: 攻击性自然法术
- **当** 德鲁伊使用攻击法术时
- **那么** 可以使用愤怒术造成自然伤害
- **并且** 可以使用月火术造成奥术伤害和持续伤害
- **并且** 可以使用星火术造成高额奥术伤害

#### 场景: 控制性自然法术
- **当** 德鲁伊使用控制法术时
- **那么** 可以使用纠缠术束缚敌人
- **并且** 可以使用自然之握将敌人拉向自己
- **并且** 可以使用飓风击退敌人

### 需求: 德鲁伊治疗能力
德鲁伊拥有独特的持续治疗专长。

#### 场景: 持续治疗法术
- **当** 德鲁伊使用治疗法术时
- **那么** 可以使用回春术提供持续治疗效果
- **并且** 可以使用愈合术快速恢复生命值
- **并且** 可以使用宁静术进行群体持续治疗

#### 场景: 治疗增强
- **当** 德鲁伊的治疗法术生效时
- **那么** 持续治疗效果可以叠加
- **并且** 自然法术威力同时影响治疗和伤害效果

### 需求: 德鲁伊召唤能力
德鲁伊可以召唤自然生物协助战斗。

#### 场景: 召唤自然生物
- **当** 德鲁伊使用召唤法术时
- **那么** 可以召唤树人协助战斗
- **并且** 可以召唤荆棘术创造障碍物

#### 场景: 自然抗性
- **当** 德鲁伊使用抗性法术时
- **那么** 可以提供自然伤害抗性
- **并且** 可以移除毒药和疾病效果

### 需求: 德鲁伊形态限制
不同形态下德鲁伊的能力有所限制。

#### 场景: 施法限制
- **当** 德鲁伊处于动物形态时
- **那么** 无法使用大部分法术技能
- **并且** 只能使用该形态专属的技能

#### 场景: 装备限制
- **当** 德鲁伊变形时
- **那么** 武器和护甲的外观效果被隐藏
- **并且** 装备属性加成仍然有效

### 需求: 德鲁伊属性需求
德鲁伊根据专精方向对不同属性有不同需求。

#### 场景: 平衡专精属性
- **当** 德鲁伊专精平衡时
- **那么** 主要需求智力和精神属性
- **并且** 智力提升法力值和法术伤害

#### 场景: 野性战斗专精属性
- **当** 德鲁伊专精野性战斗时
- **那么** 坦克形态主要需求力量和耐力
- **并且** DPS形态主要需求力量和敏捷

#### 场景: 恢复专精属性
- **当** 德鲁伊专精恢复时
- **那么** 主要需求智力和精神属性
- **并且** 智力提升治疗效果，精神提升法力恢复
